Purpose

Games are an integral part of the life of our team. We want to learn how to make games. To do this, we decided to make a 2D game to get acquainted with the basic knowledge of Game development.

Problems to solve:

- There are many games that are already successfully entertaining people. We are not going to create something different from other games, but take ideas from them and make them work properly together.

- People have a need for fun and we are going to satisfy this need with a game.

- Not a lot of work done for 2D games in UE.

Major Features

- 2D Game made in Unreal Engine

- Dynamic Story

- Free Game

- Single-player Action Linear Platformer
